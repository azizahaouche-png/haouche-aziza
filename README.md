# Practical Assignment – FOSS Tools for Academic Writing

"FOSS Tools for Academic Writing".

## Project Description
This project is a short literary analysis written in LaTeX using Overleaf.
The analysis is based on Nikolai Gogol’s novel *Dead Souls*.

The project includes:
- a LaTeX source folder downloaded from Overleaf (.zip),
- the final PDF file,
- bibliographic references managed with Zotero (BibLaTeX format).

## Tools Used
- Overleaf (LaTeX)
- Zotero (reference management)
- GitHub (sharing and version control)
